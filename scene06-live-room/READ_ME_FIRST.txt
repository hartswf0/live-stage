VILO SCENE 06 · LIVE ROOM

DUMMY-EASY SHOOT START
1. Mac: double-click START_VILO_MAC.command
   Windows: double-click START_VILO_WINDOWS.bat
2. Press START ROOM + CLIP SOUND.
3. Approve the actress camera, or deny it and keep using the remote grid.
4. Leave AUTO on. The room advances through Scene 06 by itself.

DIRECTOR
Press D or tap the invisible upper-left corner.
Each character is one row.
I = idle visual plate.
A = plate whose ORIGINAL MP4 AUDIO is used when that character speaks.
Click any card to assign both I and A.
Press SPEAK SELECTED AUDIO to test that character immediately.

SHOOT KEYS
1 / 2 / 4 / 9 = layout
SPACE = next cue
D = director table
H = clean feed
C = camera on/off

AUDIO
The app does not trust the video element for sound. Every MP4's AAC track was extracted, normalized, and embedded as its own MP3. The blue-bordered tile is the current audible character. Only one remote voice plays at a time.
