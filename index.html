Open Channels (press K) in the Full Scripted Stage: organize your 27 clips by character, preview any one with sound, grade it, and cast it. That casting is shared — every stage you open reads it, so a clip you pick in MONTE shows up in the Director Stage, the Collision stage, and the meeting itself. Repository

Every build, the shared clip manifest, and the media library — all in one place.
No installs. Open and go. Runs in browser. Chrome recommended. Works offline. Keep rolling. Host tile = actress camera. Private by default. You control access.
Media lives in media/, declared once in clips.js; each build auto-wires from it — no manual uploads. Casting and grades are shared through clips.js so MONTE joins every stage. Inside any stage, press D for director controls, K for the casting board, and H to hide chrome for a clean take.
