HOW TO USE ICARO-PRO

A dot-matrix video editor. Everything is A, B, C, D.
128×96 grid · 8 ink levels · text · motion · ripple · multi-clip

AUDIO SYNC: Video clips keep their audio. Export preserves synced soundtrack.
A
ADD: Load multiple clips into the clip bin. Camera, film leader, script files. Insert clips to timeline. Memory gauge tracks usage.
B
BUILD: Text overlay (stamp on frame ranges), motion presets (slide, fade, typewriter, scroll), ripple/wave generators, AI prompt, BEFLIX script engine.
C
COLOR: Ink directly on canvas even during playback. Pen, box, erase. 8 pressure levels.
D
SAVE: Export WebM/MP4/GIF/contact sheet. Timeline ops: split, delete, duplicate, reverse. BEFLIX code viewer.
TIMELINE: Long-press a frame to start selection. Tap another to extend. Shift+Arrow on desktop. Esc clears. Space = play/pause. 1-4 = tabs.
